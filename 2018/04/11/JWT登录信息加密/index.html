|阅读量:

1、背景

对于传统的单点登录系统，使用cookie和session的方式存储用户登录信息，但是对于安全性要求较高的企业–金融企业，就需要对用户的信息进行加密存储，防止客户信息泄露。

2、JWT构成

JWT—-JSON Web Token

第一部分我们称它为头部（header)

第二部分我们称其为载荷（payload)

第三部分是签证（signature)
